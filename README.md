# Karma Test Explorer for Visual Studio Code

The `Karma Test Explorer` extension provides visual running and debugging of Karma tests on Visual Studio Code with the
[Test Explorer UI](https://marketplace.visualstudio.com/items?itemName=hbenl.vscode-test-explorer) extension.

It is a significant rewrite based on the [Angular/Karma Test Explorer](https://github.com/Raagh/angular-karma_test-explorer) extension and focuses on providing more robust support for Karma tests (and does so without any special consideration to Angular).

It provides:

* Better support for remote development (devcontainers)
* Support for JavaScript tests (in addition to TypeScript)
* Support for providing environment variables
* Better support for code lens in much more scenarios
* _And a few other enhancements_
